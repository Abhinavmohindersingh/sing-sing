Scope of Works

What We're Building

An interactive HTML battery proposal tool — select a battery, enter site details, and generate a client-ready proposal in minutes. No Word docs. No back-and-forth.

PHASE 01
Battery Selector + Spec Auto-load
4 SKUs — PA125, PA250, PA375, PE500 — click to select
Potain specs auto-populate on selection: capacity (kWh), continuous amps, peak amps, weight, dimensions
TES + Potain co-branding throughout
Client name, project site, date — editable header fields
Data source: Potain PA/PE spec sheet
PHASE 02
Pricing Module + Discount Toggle
Hire price per week/month per SKU — editable by TES
Sale price per SKU — shown alongside hire, no second document needed
Optional discount field (% off) — for closing deals on the spot
Transport: 80km included, POA beyond — auto-displayed as exclusion
Priced in Xero · output to proposal
PHASE 03
Savings Calculator vs Diesel
Client inputs: site load (amps), daily hours, duration (weeks)
Auto-calculates: diesel generator cost vs TES battery cost
Displays: gross battery cost / fuel savings / net cost / CO₂ reduction
Visual comparison bar — the key selling point Chris described
Inspired by AMPD Energy + Ladybug formats
PHASE 04
Proposal Output + T&Cs
Print-to-PDF ready layout — professional, not a Word doc
T&C summary visible inline — key points only, not the full document
Hyperlink to full TES Battery Hire & Sale Terms — Dropbox / SharePoint / website link
Styled T&C document also redesigned and delivered as a standalone PDF
TES + Potain branding · print-ready
Stretch goal (Phase 5 — on request): A site consumption calculator where the client inputs their equipment list (crane, hoist, site office, lights) and the tool calculates their peak and continuous load demand — then recommends the right TES battery.
